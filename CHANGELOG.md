## [1.0.0] - 2025-03-10

### Added
- Ride registration with map and navigation
- Dynamically generated PDF invoices stored in Railway S3 bucket
- PDF viewer, QR code and download for invoices
- Dashboard with analytics
- JWT-based authentication with multi-device session support
- GDPR-compliant soft delete with data anonymization
- Mobile app for Android and iOS via Capacitor
- Playwright E2E test suite
- Vitest unit tests
- CI/CD pipeline via GitHub Actions
- Backend deployed on Railway, frontend deployed on Vercel
